# useful-decorators
 Some useful JavaScript stage-0 decorators

## Decorators list
- @after(fn)
- @autobind()
- @before(fn)
- @bind(context)
- @debounce(ms, options)
- @defer()
- @delay(ms)
- @immutable()
- @inject()
- @interval()
- @memoize()
- @observe(cb, options)
- @once()
- @provide()
- @singleton()
- @throttle(ms, options)
